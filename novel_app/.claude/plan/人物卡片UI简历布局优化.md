# 人物卡片UI简历布局优化

## 任务描述
把编辑人物卡片这边UI 进行一些调增： 首先顶部的 图集 点击入口删掉，然后把人物卡编辑页面做成类似简历的布局，图集照片放在角落，点击可以进入到图集查看页面， 【管理图集】 按钮删掉

## 选择的方案
方案 2：侧边栏图集布局 - 图集缩略图在右侧，基本信息在左侧，适合手机端使用

## 实施步骤

### 1. 删除顶部图集管理按钮
- 移除AppBar actions中的图集管理IconButton
- 位置：character_edit_screen.dart 第361-369行

### 2. 重构基本信息布局
- 采用侧边栏布局：左侧表单字段 + 右侧图集缩略图
- 位置：_buildBasicInfoSection() 方法
- 响应式设计：小屏幕垂直布局，大屏幕侧边栏布局

### 3. 删除现有图集区域
- 移除主体内容中的_buildGallerySection()调用
- 位置：第412-417行

### 4. 保持图集点击功能
- GalleryThumbnail保持_onTap功能
- 保留_openGallery()方法

## 技术要点
- 使用LayoutBuilder实现响应式布局
- 图集缩略图固定80px宽度
- 保持现有数据绑定和验证逻辑
- 确保手机端触摸体验

## 预期结果
- 简洁的简历式布局
- 图集缩略图作为侧边栏，不占用主要内容空间
- 完整保留图集管理功能
- 适配手机端显示
